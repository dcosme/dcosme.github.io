Comparison of University of Oregon GE Salaries

Apr 16, 2019 6 min read

load packages

library(tidyverse)
library(readxl)
library(ggridges)

define palette

area = wesanderson::wes_palette("Zissou1", 4, "continuous")

load data

data.hire = read_excel("~/Documents/GTFF/Winter Appts.xlsx") %>%
  rename("ID" = `GTFF Knack ID`,
         "FTE" = `Appt Percent`,
         "MonthlySalary" = `Monthly Salary`,
         "Department" = `Department Connection`,
         "Membership" = `Member Status (static)`)

data.dept = read_excel("~/Documents/GTFF/ListWinter2019_Acad&HiringDept.xlsx") %>%
  rename("ID" = `Knack ID`,
         "ProgramType" = `Class Desc`) %>%
  select(ID, Major, ProgramType)

data.int = data.hire %>%
  full_join(., data.dept, by = "ID")

areas = data.int %>%
  group_by(Major, Area) %>%
  summarize(n = n()) %>%
  filter(!is.na(Major)) %>%
  spread(Area, n) %>%
  mutate_if(is.numeric, funs(ifelse(is.na(.), 0, .))) %>%
  mutate(Area = ifelse(AE > HU & AE > NS & AE > SS, "AE", 
                ifelse(HU > AE & HU > NS & HU > SS, "HU", 
                ifelse(NS > AE & NS > HU & NS > SS, "NS", 
                ifelse(SS > AE & SS > NS & SS > HU, "SS", 
                ifelse(Major == "Curriculum & Teacher Education", "AE", NA)))))) %>%
  select(Major, Area)

data = data.int %>%
  select(-Area) %>%
  full_join(., areas, by = "Major") %>%
  unique()

tidy data

# check duplicate IDs
duplicates = data[duplicated(data$ID),] %>%
  arrange(ID)

data.dup = data %>% 
  filter(ID %in% duplicates$ID) %>%
  group_by(ID) %>%
  mutate(MonthlySalary1 = sum(MonthlySalary),
         FTE1 = sum(FTE)) %>%
  group_by(ID) %>% 
  slice(1) %>%
  mutate(MonthlySalary1 = ifelse(FTE1 > 50, MonthlySalary, MonthlySalary1),
         FTE1 = ifelse(FTE1 > 50, FTE, FTE1),
         MonthlySalary = MonthlySalary1,
         FTE = FTE1) %>%
  select(-c(MonthlySalary1, FTE1))

# tidy
data1 = data %>% 
  filter(!ID %in% duplicates$ID) %>%
  bind_rows(data.dup) %>%
  mutate(Level = ifelse(grepl("1", Grade), "level 1", ifelse(grepl("2", Grade), "level 2", ifelse(grepl("3", Grade), "level 3", Grade))),
         Area = ifelse(is.na(Area) & grepl("DSGN|Law", Department), "AE", 
                ifelse(is.na(Area) & grepl("Philosophy|Asian", Department), "HU", 
                ifelse(is.na(Area) & grepl("CHC", Department), "SS", Area)))) %>%
  filter(!is.na(Major))

visualize data

These data are from winter quarter 2019 and are visualized based on the department in which graduate employees (GEs) are studying. GEs are often hired in the same department in which they are pursuing their academic studies, but not always.

To zoom in on the plots, right-click on it and open the image in a new tab.

Error bars are 95% confidence intervals. If there are no error bars in the plots, that means there is no variability in salaries (either because there is a single GE or because all GEs are paid the same).

Please note that the density distributions do not represent the actual range on the x-axis. Rather, these plots should be used to compare densities across departments. Individual points are included to help illustrate the distributions. Departments with only one GE will not have a distribution.

Area codes

  • AE = Arts, Education, & Business
  • HU = Humanities
  • NS = Natural Sciences
  • SS = Social Sciences

Important note: many GEs are only guarenteed funding for 9 months, so please keep this in mind when calculating yearly salaries.
